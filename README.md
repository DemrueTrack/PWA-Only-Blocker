# PWA--Only-Blocker
Block distracting sites in normal browser tabs — installed PWAs stay accessible, forcing deliberate launches from your app menu instead of URL autofill. A typed-domain gate adds friction at PWA entry too. Each subdomain tracked independently (youtube.com ≠ music.youtube.com). Fully customizable block list. Chromium/Brave, Manifest V3.
